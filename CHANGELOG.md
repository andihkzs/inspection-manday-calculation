# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.16] - 2026-01-23

### Added
- Factory Name field in Purchase Orders section for optional factory identification
- Database support for storing factory name with calculations

## [1.0.15] - 2026-01-23

### Changed
- Automated CHANGELOG updates with version increments

## [1.0.14] - 2026-01-22

### Removed
- Removed PDF export functionality, keeping only JPG export
- Removed unused Download icon import

### Changed
- Simplified export interface with single full-width JPG export button

## [1.0.13] - Previous Release

### Added
- Initial version with AQL calculator functionality
- Multi-PO calculation support
- PDF and JPG export capabilities
- Calculation history with Supabase integration
- User authentication with password protection
- Custom naming for saved calculations
- Travel time and functional test time tracking
- Printable results with QR codes
- Privacy Policy and Terms of Service
- Help documentation
- Responsive design with sidebar navigation
